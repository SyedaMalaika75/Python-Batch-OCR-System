# Python Batch OCR System

## Overview
This project demonstrates a Python-based Batch OCR (Optical Character Recognition) system that automatically processes multiple images, extracts text (simulation), and stores the extracted text into separate files.

## Features
- Batch image processing
- OCR simulation
- Automatic text extraction
- Text file generation
- Organized folder structure

## Technologies Used
- Python
- OS Module
- File Handling
- OCR Simulation

## Folder Structure
```
project_folder/
│── images/
│── outputtext/
└── batchocr.py
```

## Learning Outcomes
- Python File Handling
- Batch Processing
- Image Processing Workflow
- OCR Fundamentals
